# specimen_database
A basic MySQL dump for a researcher to use that includes TDWG standard fields

Tables for:  
Specimen  
Species  
People  
Collecting Events  
Collectors (People/Collecting Relationship Table)  
Localities  
Authors (People/Species Relationship Table)  
Alcohol Lots
